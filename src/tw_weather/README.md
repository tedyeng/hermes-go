# tw-weather 🌦️ (台灣鄉鎮天氣預報與雨具建議 CLI)

`tw-weather` 串接中華民國交通部中央氣象署 (CWA) 氣象資料開放平臺的「鄉鎮天氣預報」API (`F-D0047`)、「今明36小時預報」API (`F-C0032-001`) 及「雨量觀測站-雨量資料」API (`O-A0002-001`)，輸入 GPS 座標、度分秒 DMS、地圖連結（Google Maps）、或直接輸入縣市鄉鎮區名稱，即可查詢未來的天氣預報、高低溫、舒適度、降雨機率，以及即時降雨量觀測，並提供精準的雨具攜帶與防汛警報。

本工具專為行動裝置終端機最佳化，**預設採用 38 欄寬度的直式排版**，非常適合在手機（如 Termius）或 Telegram 中使用。

---

## 🔧 設定與前置步驟
在使用前，請至 [氣象資料開放平臺](https://opendata.cwa.gov.tw/) 免費註冊會員，並在您的專案根目錄的 `.env` 檔案中設定 CWA API 憑證：
```env
CWA_API_KEY=您的_cwa_api_key_here
```

## 💻 使用方法與功能示範

### 1. 查詢特定地點天氣與雨具建議 (`check`)
直接傳入您要查詢的行政區或 GPS 資訊，支援多種輸入格式：
- **行政區域名稱**：例如 `"臺北市信義區"`、`"板橋區"`、`"羅東"` 等。
- **經緯度數值**：例如 `"25.0339, 121.5644"`，經緯度倒置會自動校正。
- **度分秒 DMS 格式**：例如 `"北緯25°01'33.8\" 東經121°32'37.0\""`。
- **Google 地圖網址**：支援一般網址與手機分享之 `maps.app.goo.gl` 短網址。

#### 參數說明
* `LOCATION` *(選填)*：行政區名稱、經緯度、DMS 或 Google 地圖網址（預設為 `"臺北市信義區"`）。
* `--cache-time` *(選填)*：快取過期時間（秒，預設 900 秒）。

#### 範例指令
```bash
# 查詢指定行政區
uv run tw-weather check "臺北市信義區"

# 查詢指定 GPS 座標
uv run tw-weather check "25.0339, 121.5644"
```

#### 縣市級自動降級機制 (Fallback)
若僅輸入縣市（如 `"宜蘭縣"`、`"台北"`）而未提供鄉鎮區，`check` 指令會**自動降級**調用今明 36 小時預報 API，顯示該縣市的總體預報。
```bash
uv run tw-weather check "宜蘭縣"
```

### 2. 逐小時精細預報與戶外風向警報 (`hourly`)
取得未來 48 小時內逐小時的精細預報，包含溫度、體感溫度、相對濕度、降雨機率、風速與風向。

#### 參數說明
* `LOCATION` *(選填)*：行政區名稱或 GPS（預設為 `"臺北市信義區"`）。
* `--all-hours` *(選填)*：不進行過濾，強制輸出完整的未來 48 小時。預設只顯示當日剩餘小時（若當日剩餘小於 6 小時，自動延伸顯示未來 24 小時）。
* `--cache-time` *(選填)*：快取過期時間（秒，預設 900 秒）。

#### 範例指令
```bash
# 查詢當天剩餘的逐小時氣象
uv run tw-weather hourly "臺北市信義區"

# 強制顯示未來 48 小時完整資訊
uv run tw-weather hourly "新北市板橋區" --all-hours
```

### 3. 即時降雨量觀測與未來降雨預估 (`rain`)
取得距離查詢地點最近的雨量觀測站之即時雨量觀測數據（10分鐘、1小時、3小時、24小時累積及今日日累積雨量）與未來降雨機率預估。

#### 參數說明
* `LOCATION` *(選填)*：行政區名稱、座標、DMS 或 Google 地圖網址（預設為 `"臺北市信義區"`）。
* `--lat` / `--lon` *(選填)*：直接輸入經緯度座標進行精準定位。
* `--cache-time` *(選填)*：快取過期時間（秒，預設 600 秒）。

#### 範例指令
```bash
# 查詢指定行政區最近測站雨量
uv run tw-weather rain "臺北市信義區"

# 查詢指定座標最近測站雨量
uv run tw-weather rain --lat 25.0339 --lon 121.5644
```

### 4. 縣市今明 36h 快速預報 (`summary`)
直接查詢特定縣市的今明 36 小時預報（分為 3 個 12 小時區段），顯示天氣、高低溫度、降雨機率與舒適度。

#### 參數說明
* `COUNTY` *(必填)*：縣市名稱（如 `"臺北市"`、`"高雄市"` 等，會自動校正台/臺等錯別字）。
* `--cache-time` *(選填)*：快取過期時間（秒，預設 900 秒）。

#### 範例指令
```bash
uv run tw-weather summary "臺北市"
```

一鍵下載全台 22 縣市當前時段的降雨機率與雨具建議，並依降雨機率從高到低排序，非常適合跨縣市通勤、出差或旅遊前瀏覽。

#### 參數說明
* `--cache-time` *(選填)*：快取過期時間（秒，預設 900 秒）。

uv run tw-weather board

### 6. 清除本地天氣快取資料 (`clean`)
本工具內建快取機制（將一般預報快取 15 分鐘，即時雨量快取 10 分鐘），若有即時更新需求，可手動清空 SQLite 中的快取資料。

#### 範例指令

## ☂️ 決策邏輯說明

### 雨具攜帶建議邏輯
- **降雨機率 < 30%**：`🟢 不需帶傘`
- **30% <= 降雨機率 < 70%**：`🟡 需帶折疊傘`
- **降雨機率 >= 70%**：`🔴 需帶直傘`
- **特例升級**：若天氣預報文字中包含 `"雷雨"`、`"大雨"`、`"豪雨"`、`"暴雨"` 等劇烈天氣詞彙，即使降雨機率未達 70%，建議也會自動升級為 `🔴 需帶直傘` 以策安全。

### 風速警報標準
- 風速 $\ge 10.8\text{ m/s}$ (達 6 級風以上) 時，風速數值會以紅色醒目顯示，提示強風危險。
- 一般風速以綠色或普通顏色顯示。

### 即時雨勢體感標準
- $0.1 \sim 2.0 \text{ mm/hr}$ $\rightarrow$ `毛毛雨，可不撐傘`
- $2.0 \sim 10.0 \text{ mm/hr}$ $\rightarrow$ `小雨，出門記得帶傘`
- $10.0 \sim 30.0 \text{ mm/hr}$ $\rightarrow$ `大雨，開車騎車請減速`
- $> 30.0 \text{ mm/hr}$ $\rightarrow$ `豪雨或雷雨，注意低窪地區積水`
- $\text{rate} \le 0.0$ $\rightarrow$ `目前無顯著降雨`

### 未來預估雨量標準 (綜合 Wx 與 PoP 評估)
- **無雨**：降雨機率為 $0\%$，或降雨機率 $< 30\%$ 且預報無雨字樣。
- **微量/毛毛雨 (<2 mm)**：預報為毛毛雨、雨跡或短暫雨等。
- **小雨至中雨 (約 2~10 mm)**：預報為陣雨或一般降雨，或降雨機率 $\ge 70\%$ 但無劇烈天氣字樣。
- **大雨等級 (約 10~30 mm)**：預報包含大雨、雷雨或雷陣雨。
- **豪雨等級 (約 >30 mm)**：預報包含暴雨、豪雨或大豪雨等劇烈大雨。


### 防災雨量警戒標準 (24H 與 3H 累積量)
- **大雨 (Heavy Rain)**：24H 累積雨量 $\ge 80\text{ mm}$ 或 1H 累積雨量 $\ge 40\text{ mm}$。
- **豪雨 (Extremely Heavy)**：24H 累積雨量 $\ge 200\text{ mm}$ 或 3H 累積雨量 $\ge 100\text{ mm}$。
- **大豪雨 (Torrential)**：24H 累積雨量 $\ge 350\text{ mm}$ 或 3H 累積雨量 $\ge 200\text{ mm}$。
- **超大豪雨 (Extremely Torrential)**：24H 累積雨量 $\ge 500\text{ mm}$。
